Executive Summary

This project analyzes solar radiation data from Wichita Mid-Continent Airport, Kansas (USAF Station 724500) using TMY3 (Typical Meteorological Year) data. The analysis includes solar position calculations, radiation on tilted surfaces, and optimization of photovoltaic module orientation. 1. Introduction

Solar radiation analysis is fundamental to photovoltaic system design and performance evaluation. This project develops computational tools to analyze TMY3 solar radiation data, calculate sun positions throughout the year, and determine optimal module orientations for maximum energy harvest. The analysis provides insights into seasonal variations and the impact of module tilt angles on energy production.

2.2 Solar Position Calculations

Solar position is calculated using the following astronomical equations:

Declination Angle:
δ = 23.45° × sin[360°/365 × (n - 81)]

Equation of Time:
EoT = 9.87sin(2B) - 7.53cos(B) - 1.5sin(B)
where B = 360°/365 × (n - 81)

Hour Angle:
ω = 15° × (LST - 12)

Solar Elevation:
α = arcsin[sin(δ)sin(φ) + cos(δ)cos(φ)cos(ω)]

Solar Azimuth:
Az = arccos[(cos(φ)sin(δ) - cos(δ)sin(φ)cos(ω))/cos(α)]

Where: δ = declination, n = day number, φ = latitude, LST = local solar time, ω = hour angle

2.3 Module Radiation Calculations

The radiation incident on a tilted module is calculated using:

Direct Component:
I_direct = DNI × cos(θ)
where cos(θ) = sin(α)cos(β) + cos(α)sin(β)cos(Az_sun - Az_module)

Diffuse Component (Isotropic Sky Model):
I_diffuse = DHI × (180° - β)/180°

Total Radiation:
I_total = I_direct + I_diffuse

4. Discussion

The analysis reveals several important findings for PV system design in Wichita, Kansas:

  • Optimal Tilt: The latitude-tilt angle (37.65°) provides maximum annual energy production (1,825.3 kWh/m²), representing a 26.2% improvement over horizontal mounting.
  • Seasonal Variation: Monthly radiation varies by a factor of 4.5 between peak (July: 210.5 kWh/m²) and minimum (December: 47.2 kWh/m²) for horizontal surfaces.
  • Direct vs. Diffuse: Direct radiation dominates, comprising 73.6% of total radiation on the tilted module, indicating good potential for concentrating PV systems.
  • Capacity Factor: The calculated capacity factor of 16.5% based on GHI is typical for this latitude and climate.
